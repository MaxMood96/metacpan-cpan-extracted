# Data::Smile

[![PP](https://github.com/tobyink/p5-data-smile/actions/workflows/PP.yml/badge.svg)](https://github.com/tobyink/p5-data-smile/actions/workflows/PP.yml)
[![XS](https://github.com/tobyink/p5-data-smile/actions/workflows/XS.yml/badge.svg)](https://github.com/tobyink/p5-data-smile/actions/workflows/XS.yml)

`Data::Smile` is a Perl distribution for reading and writing Smile, a compact binary JSON-like format. It provides XS and Pure Perl implementations with tests, plus examples for encoding and decoding data. Use it when you need interoperable Smile support in Perl applications.
