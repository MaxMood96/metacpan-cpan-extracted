=encoding utf8

=head1 NAME

OODoc::Format::Pod2 - Produce POD pages from the doc tree with a template

=head1 INHERITANCE

 OODoc::Format::Pod2
   is an OODoc::Format::Pod
   is an OODoc::Format
   is an OODoc::Object

 OODoc::Format::Pod2
   is an OODoc::Format::TemplateMagic

=head1 SYNOPSIS

  my $doc = OODoc->new(...);
  $doc->formatter('pod2')->createPages(
     manual_options => [ show_examples => 'NO' ],
  );

=head1 DESCRIPTION

Create manual pages in the POD syntax, using the Template::Magic
template system.  It may be a bit simpler to configure the outcome
using the template, than using L<OODoc::Format::Pod|OODoc::Format::Pod>, however you
first need to install L<Bundle::Template::Magic>.

Extends L<"DESCRIPTION" in OODoc::Format::TemplateMagic|OODoc::Format::TemplateMagic/"DESCRIPTION">.

Extends L<"DESCRIPTION" in OODoc::Format::Pod|OODoc::Format::Pod/"DESCRIPTION">.

=head1 DIAGNOSTICS

=over 4

=item Error: cannot link to a $package

Cast by link()

=item Fault: cannot read prelimary pod from $file: $!

Cast by simplifyPod()

=item Fault: cannot write final pod to $file: $!

Cast by simplifyPod()

=item Error: chapter without name in template

In your template file, a C<{chapter}> statement is used, which is
erroneous, because it requires a chapter name.
Cast by templateChapter()

=item Error: illegal show option choice: $value

Cast by showOptions()

=item Error: illegal value for show_option_table: $value

Cast by showSubroutine()

=item Error: illegal value for show_sub_description: $value

Cast by showSubroutine()

=item Error: illegal value for show_subroutine: $value

Cast by showSubroutine()

=item Error: missing required chapter $name in $manual

Cast by showRequiredChapter()

=item Warning: no meaning for container $tags in chapter block

Cast by templateChapter()

=item Fault: write to $file failed: $!

Cast by simplifyPod()

=back
