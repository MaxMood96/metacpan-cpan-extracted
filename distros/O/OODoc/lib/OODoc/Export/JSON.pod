=encoding utf8

=head1 NAME

OODoc::Export::JSON - Dump the parsed docs into JSON

=head1 INHERITANCE

 OODoc::Export::JSON
   is an OODoc::Export
   is an OODoc::Object

=head1 SYNOPSIS

  my $doc = OODoc->new(...);
  $doc->export('json');

  my $exporter = OODoc::Export::JSON->new;

=head1 DESCRIPTION

Create a JSON dump or the parsed documentation, useful to work with dynamically
generated web-pages.
