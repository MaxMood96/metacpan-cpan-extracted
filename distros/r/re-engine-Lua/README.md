re-engine-Lua
=============

[![Licence](http://img.shields.io/badge/Licence-MIT-brightgreen.svg)](LICENSE)

INSTALLATION
------------

To install this module type the following:

    perl Makefile.PL
    make
    make test
    make install

DEPENDENCIES
------------

Needs Perl 5.10 or later for pluggable regex support.
